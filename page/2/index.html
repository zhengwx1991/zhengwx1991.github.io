Linux安装RabbitMQ

Linux安装RabbitMQ

安装Erlang
下载包
    wget http://www.rabbitmq.com/releases/erlang/erlang-19.0.4-1.el7.centos.x86_64.rpm
解压包
    rpm -ivh erlang-19.0.4-1.el7.centos.x86_64.rpm
安装
    yum -y install erlang
查看版本，验证是否安装成功
    erl -version
