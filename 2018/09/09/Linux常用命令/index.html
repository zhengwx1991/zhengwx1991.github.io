Linux

Linux常用命令

文件操作：
// 打开文件夹    
cd /home/admin
// 返回上一级
cd ..

// 创建文件夹
mkdir xxx

// 复制文件
cp 要复制的文件名 目标目录

// 剪切文件
mv 要复制的文件名 目标目录

// 删除文件夹命令
rm -rf 要删除的文件夹名（注：-f为强制删除，-r为文件夹）

// 删除文件命令
rm -f 要删除的文件名
